# ServiceNow Incident Automation (Zurich)

## Overview
This project demonstrates an automated incident-routing workflow built using ServiceNow Flow Designer in a Zurich Personal Developer Instance.

## Business Problem
Manual triage of security-related incidents caused delays and inconsistent assignment.

## Solution
- Implemented keyword-based logic to identify security-related incidents
- Automated assignment to the Security Operations group
- Updated incident state automatically using Flow Designer

## Technologies Used
- ServiceNow Zurich
- Flow Designer
- Incident Management

## Notes
Due to licensing restrictions in ServiceNow PDIs, this solution demonstrates security incident handling using out-of-box Incident Management and Flow Designer automation.
